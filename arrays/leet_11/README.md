## 11. Container With Most Water

Link: https://leetcode.com/problems/container-with-most-water/description/

-------------------------
### Difficulty : Medium

### Approach

- We use a double pointer approach. We begin with the extreme a go towards the middle with our two pointers. We advance the shortest of the two.

## Time complexity

O(n). We only visit each element once.

## Space complexity

O(1). Only some variables are used
